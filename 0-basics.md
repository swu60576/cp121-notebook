## basics
> เราจะพูดถึงเรื่องเบสิคสำคัญๆ สองเรื่อง ได้แก่ เรื่องของการประกาศตัวแปร กับ เรื่องของลำดับในการคำนวน เช่น

ทำไม
```javascript
a = 5;
b = 6;
cSum = cSum + a + b;
```
ถึงเกิด error

และ 

ทำไม a กับ b ด้านล่างนี้ถึงไม่เท่ากัน
```javascript
a = 1 / 5 * 2;
b = (1 / 5 ) * 2;
write('a :' + a + ' b: ' + b); 
```
